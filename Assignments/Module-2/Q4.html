Q-4 How to used JavaScript Events to do all examples?

Ans:-


Event handlers can be used to handle and verify user input, user actions, and browser actions: Things that should be done every time a page loads. Things that should be done when the page is closed. Action that should be performed when a user clicks a button.

or

JavaScript Events are many parts in devided. like-

1-) Mouse events (e.g., click, mouseover, mouseout)

2-) Keyboard events (e.g., keydown, keyup)

3-) Form events (e.g., submit, change, focus)

4-) Document events (e.g., load, unload)

5-) Window events (e.g., resize, scroll)

6-)Touch events (e.g., touchstart, touchmove, touchend)
